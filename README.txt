Dependencies to run the file : ------------------------------

1) bokeh
   command (Ubuntu) : pip install bokeh
2) numpy
   command (Ubuntu) : sudo apt-get install python-numpy

"Detailed Plot" tab is just shown as representation for the passbands at 5500K

Also created a new tab representing "Transmissions" representing the Transmission vs Wavelength curve, based on data of Besell approximations for UBVR passbands

Also added hover options to view any point on these plots.
